Documentação da API

A API da Pickles Analytics permite integrar nossa inteligência de dados diretamente em suas aplicações. Acesse dados de milhões de empresas, monitore licitações e enriqueça seu CRM programaticamente.

A API está disponível apenas para clientes do plano Enterprise. Entre em contato para obter sua chave de acesso.

Autenticação

Todas as requisições devem incluir sua chave de API no cabeçalho X-API-Key.

bash 
curl -X GET "https://api.pickles.com/v1/status" \
  -H "X-API-Key: sua_chave_aqui"

Consulta CNPJ

Retorna detalhes completos de uma empresa, incluindo sócios, capital social e situação cadastral.

GET /v1/company/{cnpj}
JSON Response 
{
  "cnpj": "00.000.000/0001-91",
  "razao_social": "BANCO DO BRASIL SA",
  "nome_fantasia": "DIRECAO GERAL",
  "situacao": "ATIVA",
  "capital_social": 67000000000.00,
  "endereco": {
     "uf": "DF",
     "municipio": "BRASILIA"
  }
}
